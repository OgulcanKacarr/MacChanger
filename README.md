# MacChanger
basic mac changer for Kali

```
usage: mac_changer.py [-h] -i INTERFACE [-r RANDOM] [-m--mac M__MAC]

options:
  -h, --help            show this help message and exit
  -i INTERFACE, --interface INTERFACE
                        interface
  -r RANDOM, --random RANDOM
                        random mac
  -m--mac M__MAC        manuel mac

```
